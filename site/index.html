In Dazzle the input's hx-get hits /app/command, which returns persona-scoped results. Here a mock htmx returns a canned list so the demo works with no server.

Confirm dialoginteractive

Designed replacement for window.confirm — every hx-confirm upgrades automatically.

<button class="dz-button dz-button-destructive" hx-delete="/mock/noop" hx-confirm="Delete this invoice? This cannot be undone.">Delete invoice</button>
dz-confirm.js intercepts htmx:confirm. No per-button wiring — any element with hx-confirm gets the designed dialog.
